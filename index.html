Data Methodology & Disclaimer

All data is sourced from SimilarWeb's Global Gen AI Tracker and reflects web browser traffic only. This data does not include desktop application usage (e.g. Claude Desktop, ChatGPT desktop app), mobile-native app traffic, API calls, or AI embedded within other products. MAU (Monthly Active Users) figures are estimates based on web traffic modeling. Actual total usage across all platforms will be significantly higher than what is shown here.
